# HEALTHTECH

## Sobre o projeto


O cenário analisado apresenta uma cuidadora de idosos autônoma que atende diferentes pacientes em domicílio. As informações importantes sobre os atendimentos ficam dispersas entre anotações, mensagens e memória, o que dificulta a organização da rotina e a consulta rápida do histórico de cada paciente.

Diante desse contexto, o problema identificado foi a **falta de centralização das informações dos pacientes e dos registros de atendimento**.

A solução proposta foi a criação de uma **API backend** capaz de:

- cadastrar pacientes;
- registrar atendimentos domiciliares;
- consultar o histórico individual de cada paciente;
- organizar informações de forma estruturada para facilitar o acompanhamento e a comunicação com familiares.
